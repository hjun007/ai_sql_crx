# 图标文件说明

请在此目录下放置以下尺寸的 PNG 图标文件：

- `icon16.png` - 16x16 像素（用于扩展程序菜单）
- `icon48.png` - 48x48 像素（用于扩展程序管理页面）
- `icon128.png` - 128x128 像素（用于 Chrome 网上应用店）

## 图标要求

- 格式：PNG
- 背景：建议使用透明背景
- 设计：简洁明了，易于识别

## 在线图标生成工具

你可以使用以下工具生成图标：
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://www.canva.com/

## 临时解决方案

如果没有图标文件，插件仍然可以运行，Chrome 会显示默认的扩展程序图标。
